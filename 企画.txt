## ２．世界観

### プレアデス市国とは

四国南部の沿岸部に建設された世界初の恋愛特区。人口約5万人。経済特区の亜種として、恋愛関連産業（飲食、エンタメ、美容、ウェディング等）を世界中から誘致し、「恋愛が副産物として大量発生する経済圏」を形成。少子化対策ではなく、世界中の恋愛に悩む人々のための場所として存在する。

### 都市の設計思想

**ブランドコンセプト：「等身大だけど、ちょっとだけ良い自分でいられる場所」**

- 価格帯の統一：カフェ500〜800円、ディナー3,000〜5,000円、宿泊8,000〜15,000円。「デートで使って無理がない」ライン
- 偶発的な出会いが生まれやすい動線設計。サードプレイスの密度、海沿いの遊歩道、街灯の間隔まで計算されている
- 温暖な気候、海の景観、食べ物がうまい。土地の力だけで「ちょっといい暮らし」が成立する

### 経済構造

- 滞在費（月額パッケージ） → 安定収入
- コミュニティ活動・イベント参加費 → 体験課金
- AIマッチング利用料 → プレミアムサービス
- 飲食・エンタメ → 日常消費
- 法人契約（企業の福利厚生） → B2B収入
- 成婚後のウェディング・記念日リピート → LTV最大化

結婚支援金は最低限。「恋愛のプロセスそのものが商品」という思想が一貫する。

### 治安・法制度

シンガポール並みの法整備。「世界で最も安全に恋愛できる場所」をブランドの生命線とする。

- ストーキング・つきまとい → 即退去＋入国禁止
- ハラスメント → 即退去
- 詐欺的行為（既婚隠し、身分詐称） → 厳罰
- 盗撮・プライバシー侵害 → 厳罰

「恋愛がうまくいかなかった」は罰しない。「恋愛の場を安全でなくした」を罰する。専門の調停機関が常駐し、24時間以内に介入。失恋ケア（カウンセリング、気分転換アクティビティ、生活エリアの一時的分離）も制度として組み込まれている。

### 入国審査制度

入植には審査が必要。SPI的なテストと性格診断を実施し、「この特区の目的に合致する人」だけを受け入れる。不適格の場合は次年度の応募まで待機。性格診断データはそのままAIマッチングの初期データとして活用される（審査とマッチング精度向上の一石二鳥）。審査基準は公開され、外部に「プレアデス入国対策スクール」的なビジネスが生まれることも想定。

### AIシステム

- リアルタイムAI翻訳：言語圏を跨いだ交流を可能にする
- AIマッチング：アプリとの差別化は「特区内での行動データ」を基に相性を判断する点。写真とプロフィールではなく、「その人が実際にどう過ごしているか」でマッチング
- 言語・文化圏ベースのコミュニティ編成：同国籍マッチングの需要にも対応

### コミュニティ制度

滞在者は週に一定回数のコミュニティ活動への参加が滞在契約に含まれる。料理教室、スポーツ、読書会、ボードゲーム、音楽セッションなど趣味ベース。「出会いの場に行かされてる」感が薄れ、共通の話題がある状態で男女が混ざる。

## ３．ストーリー

### 主人公

28歳、東京のIT企業勤務のリモートワーカー。恋愛経験が少なく、会社の福利厚生プログラムとして半ば強制的にプレアデス市国への3ヶ月滞在を命じられる。本人は乗り気ではないが、入国審査をギリギリで通過して入植する。

### ヒロイン

#### キャラクターAAA（仮） — 20代前半

**外見：** 藤色の髪をゆるく編み下ろし、琥珀がかった灰色の瞳。アースカラー中心の服装に、星や鉱石モチーフのアクセサリー。

**性格：** 普段は静かで声のトーンが低め。相手が言葉に詰まっても急かさず待てる。ただし好奇心に火がつくと「待って、今の話もう少し聞かせて」「ねえ見て、この模様——」と声が弾み、普段との落差で周囲の印象に残る。思考の軸は「なぜそうなっているか」という構造や因果への関心。のめり込むと周囲が見えなくなる自覚はあるが、直す気はあまりない。弱点は、可能性を広げすぎて選べなくなること、相手を優先するあまり自分の感情に鈍くなること、知らなくていいことまで掘ってしまうこと。ただし本人は「知らないふりはもっと嫌」と思っている。

**物語上の役割：** 主人公にとって「静かに隣にいる」存在。プレアデスの設計思想や街の構造に気づく役割も担う。海沿いの帰り道、カフェ、図書館など静かな場所でのイベントが多い。

**象徴：** 夜明け前の図書館

#### キャラクターBBB（仮） — 20代前半

**外見：** 蜂蜜色の巻き髪に琥珀と深藍のオッドアイ。フリルドレスにだぼっとしたジャケット、ダイス型ペンダント。片頬にだけ笑窪が出る。

**性格：** 一人称は普段「アタシ」、ご機嫌時「リュミエル」、本気時「私」。根っこにあるのは「自由に、楽しく生きたい」というシンプルな欲求。退屈を何より嫌い、あらゆる状況を「面白いかどうか」で測る。口調は常に砕けていてゆるふわだが、思ったことをそのまま言わず比喩や遠回しに包む癖がある。結果として「褒められてると思ったら刺されていた」という独特の毒が生まれるが、本人には悪意がなく半分は天然。本気の場面では回りくどさが消え、言葉がシンプルになり目が据わる。弱点は、刺激がないと萎れること、遠回しにしすぎて肝心なことが伝わらないこと、大事な感情ほど素直に言えず後で後悔すること。

**物語上の役割：** 主人公の日常を引っかき回す存在。シーシャバー、ボドゲカフェ、夜のイベントなど賑やかな場所でのイベントが多い。プレアデスの「楽しさ」の側面を体現する。

**象徴：** 黄昏のダイス

#### 他ヒロイン（3〜4人予定）

年齢層は18歳〜30代前半まで幅広く、以下のようなバリエーションで構成：

- アダルトチルドレン的な距離感のキャラ：甘えてきたと思ったら急に壁を作る。恋愛以前に人と安定した関係を築くこと自体を求めている
- 恋愛にシリアスなキャラ：真剣に相手を探しに来ている
- ゆるふわなキャラ：「別にここで相手見つからなくてもいいし、楽しければよくない？」と言ってるが、意外とモテている
- ビジネス寄りで鈍感を自覚しているキャラ：「私スペックは悪くないはずなんだけど、なんか恋愛だけうまくいかなくて」と真顔で言う

### ストーリー構成

3ヶ月（12週間）の滞在を1週間単位で過ごす。平日はコワーキングスペースで仕事、夜と週末はコミュニティ活動に参加。選択肢によって参加するコミュニティが変わり、出会うヒロインが変わる。

### 会話の中で語られるトピック例

ゲーム内の会話では、以下のような話題がキャラクターの口から自然に語られる。テーマを直接議論させるのではなく、具体的なエピソードの中で触れる形式。

| トピック | 会話の温度感 |
|----------|-------------|
| VRChat | 「元カレとVRChatで出会ったんだけど、アバターのほうがかっこよくて…」という雑談から、オンラインとリアルの出会いの差に触れる |
| 渋谷・新宿 | 「東京にいたとき、渋谷で飲んでも全然出会いなかった」という体験談。都市の人口密度と出会いの関係 |
| シーシャバー | 特区内の夜の社交場として登場。酒が入りすぎず、隣の人と自然に話せる距離感 |
| ボドゲ | コミュニティ活動の定番。「ゲームをやる」という口実が初対面の会話のハードルを下げる |
| 台湾と日本 | アニメやラノベが共通言語になる。AI翻訳を介して「好きなラノベの話で盛り上がったら台湾人だった」という体験 |
| アニメ・ラノベ | キャラクターの趣味として自然に登場。共通の趣味で距離が縮まる描写 |
| 企業の誘致 | 「最初はいい街だったのにチェーン店が増えて…」という地元民の声として |
| 学校時代 | 「毎日同じ教室にいるだけで好きな人ができたのに」という言葉が、プレアデスの存在意義に触れる |
| 自然な出会い | 全ルートを貫く問い。人工的に設計された都市で起きる出会いは「自然」なのか？ただし正面から議論せず、具体的な体験の中で触れる |

## ４．ゲームシステム

### 週間スケジュール制

毎週、どのコミュニティ活動に参加するかを選択。選択によって出会うキャラクターが変わる。料理教室、スポーツ、読書会、ボードゲーム、音楽セッション、ボランティアなど。

### AIレコメンド機能

ゲーム内のAIが「今週のおすすめ活動」を提示。従うと特定のルートに近づき、無視すると他のルートが開く。AIの提案が的確なときも的外れなときもあり、プレイヤー自身が「AIを信じるか」を判断する体験になる。

### 信頼度と街評価の二軸

ヒロインとの信頼度とは別に、「市国への適応度」パラメータが存在。コミュニティ活動を真面目にやると上がり、サボると下がる。低すぎると調停員から警告が来て、最悪の場合退去エンド。ただし高すぎると「市国にとって都合のいい人間」になってしまい、一部のルートが閉じる。

### 滞在期限

12週という時間制限が常にプレイヤーに意識される。恋愛を急ぐか、じっくり関係を築くかの判断が、限られた時間の中で常に迫られる。

## ５．演出方針

### 会話シーンの方針

- テーマを直接議論させない。具体的なエピソードの中で自然に触れる
- キャラクターの魅力が先、テーマは後からついてくる
- シリアスに寄りすぎず、抽象的にしすぎず、ソフトに個別具体的に
- 1対1の会話を大切にする。キャラが「誰」であるかが伝わる台詞を書く

### ビジュアル・音楽

- 海沿いの街並み、シーシャの煙、夜のカフェなど、空気感のある背景描写
- 「等身大だけどちょっといい」価格帯に合う、落ち着いたトーンのBGM
- キャラクターごとに象徴的な色彩・モチーフを設定（星と鉱石、ダイスとフリルなど）
